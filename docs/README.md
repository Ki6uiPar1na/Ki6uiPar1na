## 🎨 Design System

- **Primary Color**: `#00FF41` (Matrix Green)
- **Secondary Color**: `#00D4FF` (Cyber Blue)
- **Accent Color**: `#FFD700` (Gold)
- **Background**: `#0A0A0A` (Deep Black)
- **Glassmorphism**: Semi-transparent cards with green borders
- **Typography**: Fira Code / Courier New (Monospace)

## 🚀 Features

- Custom SVG assets with neon glow effects
- Matrix rain and grid overlay aesthetics
- Terminal-style command prompt sections
- Glassmorphism card components
- Auto-updating GitHub Actions workflows
- Responsive layout across devices
- CTF achievement timeline visualization

## 🔧 Setup

1. Fork this repository
2. Replace `Ki6uiPar1na` with your GitHub username in README.md
3. Update links, emails, and usernames for your profiles
4. Enable GitHub Actions
5. Customize SVG assets in `assets/` directory

## 📊 GitHub Actions

| Workflow | Schedule | Description |
|----------|----------|-------------|
| `snake.yml` | Daily | Generates contribution snake animation |
| `metrics.yml` | Daily | Updates GitHub statistics metrics |
| `activity.yml` | Every 6h | Updates recent activity section |
| `achievements.yml` | Weekly | Refreshes trophy and achievement badges |
